# Perchance App Engine — Firefox Extension

Turn any public perchance.org generator into its own popout desktop app with custom JS injectables & secure memory profiles!

## 📦 How to Install This Extension in Mozilla Firefox

Follow these quick steps to load this temporary developer extension:

1. **Download and Extract**:
   Extract the downloaded zip file (`perchance-firefox-extension.zip`) into a folder on your computer. You should see files like `manifest.json`, `popup.html`, and `background.js` inside.

2. **Open Developer Debugging Page**:
   In your Firefox browser, open a new tab and go to:
   `about:debugging`

3. **Navigate to This Firefox Area**:
   In the left sidebar, click on **"This Firefox"**.

4. **Load Temporary Add-on**:
   Scroll to the "Temporary Extensions" section and click the **"Load Temporary Add-on..."** button.

5. **Select Manifest File**:
   In the file chooser, navigate to the extracted folder, select the `manifest.json` file, and click open.

6. **Pin the App Engine Extension**:
   Click the Extensions (puzzle piece) icon to the right of your Firefox address bar, find **Perchance App Engine**, and pin it to your toolbar.

***

## 🚀 How to Use

1. Click the **Perchance App Engine** extension icon on your Firefox toolbar.
2. In the Launcher panel:
   - Click **+ Add App** to register new perchance generators (enter the slug, name, and desc).
   - Click **Global JS** to write persistent scripts that run across all generators upon load.
   - Click **Edit** on any card to edit its description, colors, and assign individual javascript overrides.
   - Click **Launch** to launch the generator in its own borderless window! It runs independently!

Enjoy decoupled, high-performance, ad-free Perchance generators!
